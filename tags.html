This is the anchor (a) tag. It makes the text inside, a clickable link.

Absolute Links are generally used to send readers of your web site to another web site. Absolute links start with http://

Relative links do not start with http:/. Instead, they're paths with respect to the file system.

Heading Tags <h1-6></h1-6>

Heading Tags aren't just for creating different sized titles. Heading tags help the internet browser know which section of your page is the most important. This also helps in the SEO process when setting up a web page. Never mix the h tags in the flow. If you start with h1 then the next should be h2 and so on and so on. 

            
<h1>I am very very important.</h1>
<h2>I am very important.</h2>
<h3>I am important.</h3>
<h4>I am kind ofimportant.</h4>
<h5>I am somewhere below important.</h5>
<h6>I am more important than a p tag.</h6>

Image Tags <img src="locationofimage" alt="Text for Disabilities">

The image tag needs a src attribute to provide the URL (absolute or relative) of where its located. Image tags are empty tags in that they have no content and, therefore, no end tag. A required attribute for the image tag is the "alt" attribute. The value of the "alt" attribute is displayed if an image fails to load. 

            
<img src="locationofimage" width="200px" alt="in case
the img fails to load">

display flex

.container(Main Parent) -->
.container .display-flex(Flex Parent) -->
.flex-child(Children inside(Grey Boxes))

With display flex, elements inside want to "shrink" down to as small as they can get. 

                    
.container-flex{
display: flex;
}

As you can see above without any height or width, the children aren't the same size.

You can either hard code the width and height or... you can use the following below to have flex-box automatically stretch to fit the parent container. 

                    
.container-flex > *{
    flex-basis: 100%;
}

Remember not to place this on the direct parent .container-flex, but rather, place this on the direct children of .container-flex.

.container-flex > *{ } or... ALL direct children OF .container-flex This will then make all the blocks the same width. Allowing the blocks to stretch to 100% as indicated in the example code.

justify-content center

With the code below we can "CENTER" the blocks by adjusting the dircect child, in this case the child is .step-3.

.step-3 is the class="" name for the .container-flex.

.container(Main Parent) -->
.container-flex(Flex Parent) -->
.flex-child(Children inside(Grey Boxes)) 

                    
.container .step-3{
    justify-content: center;
}

align-items center

Notice that the difference between ex 3.1 and the others is the height added in this one. Without the height, align-items: center; does not work.

justify-content --> aligns the horizontal axis

align-items --> aligns the vertical axis.

.container(Main Parent) -->
.container-align .align(Flex Parent) -->
.flex-child(Children inside(Grey Boxes)) 

                    
.container-align .align{
    height: 20rem;
    justify-content: center;
    align-items: center;
}

Flex-start

In this example we have to seperate containers:

- (top) flex-start
- (bottom) flex-end

This can be easily accomplished as shown below.

.container(Main Parent) -->
.container-flex .flex-start(Flex Parent) -->
.flex-child(Children inside(Grey Boxes))

.container(Main Parent) -->
.container-flex .flex-end(Flex Parent) -->
.flex-child(Children inside(Grey Boxes)) 

                    
.container .flex-start{
    justify-content: flex-start;
}

.container .flex-end{
    justify-content: flex-end;
}
